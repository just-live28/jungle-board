# [Jungle-Board]
일주일 동안 Next.js를 학습하며, 게시글 및 댓글 기능이 구현된 게시판을 만들어 보았습니다.

Next.js로 백엔드-프론트엔드를 구현하였고, 데이터베이스로는 MongoDB를 사용했습니다.
